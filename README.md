## Objective
The objective of this project is to analyze Netflix movies and TV shows
using Power BI to understand content trends, audience engagement, and
performance based on IMDb ratings, votes, runtime, and certifications.

## Dataset
The dataset contains information on Netflix titles, including:
- Title type (Movie / TV Show)
- IMDb ratings and vote counts
- Runtime (in hours)
- Release year
- Age certification

The raw data was provided in Excel format and transformed in Power BI.

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Data Modeling

## Analysis Performed
- Movies vs TV shows comparison
- IMDb ratings and votes trend analysis over time
- Identification of top-rated titles
- Runtime distribution and trend analysis
- Certification-wise rating comparison
- Content volume and engagement analysis across release years

## Key Insights
- Movies form a larger share of Netflix content compared to TV shows
- Viewer engagement (IMDb votes) increases significantly for recent releases
- Higher-rated content is concentrated within specific certifications
- Runtime patterns vary across time periods and content types
- A small set of titles consistently achieves higher ratings and votes


## Conclusion
This project demonstrates exploratory data analysis using Power BI,
highlighting the ability to transform entertainment data into clear,
interactive insights through effective visual storytelling.
